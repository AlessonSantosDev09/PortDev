Minhas Habilidades

HTML

Linguagem padrão para criação de páginas web. Define a estrutura e o conteúdo de uma página. Utiliza elementos e tags para formatar o texto, inserir imagens, criar links e organizar o layout da página.

CSS

Controle da apresentação visual de páginas da web, permitindo definir layout, cores, tipografia e outros aspectos do design de um site.

JavaScript

É uma linguagem baseada em protótipos, multi-paradigma e dinâmica, suportando estilos de orientação a objetos, imperativos e declarativos (como por exemplo a programação funcional).

Bootstrap

Construção de layouts responsivos,desenvolvido para ser mobile-first. Estilização de elementos da página. Imagens responsivas com redimensionamento automático.
